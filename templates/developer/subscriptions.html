API Reference

Subscriptions API

Subscriptions manage the billing relationship between a customer and a plan. They transition through status states as payments are processed or fail.

Enroll a Subscriber

POST /api/subscriptions

Initiates a new subscription enrollment. Generates and returns a checkout_link (hosted by Nomba) to collect the subscriber's payment card.

Request Body (JSON)
{ "plan_id": "e9c0b2cd-9e81-4d32-bc51-0a12e34f5678", "customer_email": "customer@example.com", "customer_name": "Emeka Eze", "callback_url": "https://yoursite.com/signup-complete" }
Response JSON (201 Created)
{ "id": "sub_48f61c20-7dbe-4...", "plan_id": "e9c0b2cd-9e81-4d32-bc51-0a12e34f5678", "customer_email": "customer@example.com", "customer_name": "Emeka Eze", "status": "active", "current_period_start": "2026-07-01T12:00:00Z", "current_period_end": "2026-07-31T12:00:00Z", "checkout_link": "https://pay.nomba.com/sandbox/QMojVVI..." }

Get Subscription Details

GET /api/subscriptions/{subscription_id}

Look up a subscription's current state, tokenized card keys, and scheduled retries.

Response JSON (200 OK)
{ "id": "sub_48f61c20-7dbe-4...", "plan_id": "e9c0b2cd-9e81-4...", "customer_email": "customer@example.com", "customer_name": "Emeka Eze", "status": "active", "current_period_start": "2026-07-01T12:00:00Z", "current_period_end": "2026-07-31T12:00:00Z", "token_key": "token_key_abc123xyz", "retry_count": 0, "next_retry_at": null, "cancelled_at": null, "cancel_at_period_end": false }

Cancel Subscription

POST /api/subscriptions/{subscription_id}/cancel

Cancels a subscription. By default, cancellation is immediate. You can optionally schedule the cancellation for the end of the current billing cycle.

Request Body (JSON) - Optional
{ "cancel_at_period_end": true }
Response JSON (200 OK)
{ "message": "Subscription scheduled to cancel at period end", "status": "active", "cancel_at_period_end": true }
