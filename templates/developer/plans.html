API Reference

Plans API

Plans represent recurring billing schedules (such as monthly subscriptions, weekly services, or custom intervals) that you define and expose to your customers.

Create a Plan

POST /api/plans

Define a new billing package. The interval is defined explicitly in days (e.g., 30 for standard monthly).

Request Body (JSON)
{ "name": "Standard Monthly", "amount": 2000.00, "currency": "NGN", "interval_days": 30, "trial_days": 0 }
Response JSON (201 Created)
{ "id": "e9c0b2cd-9e81-4d32-bc51-0a12e34f5678", "name": "Standard Monthly", "amount": 2000.00, "currency": "NGN", "interval_days": 30, "trial_days": 0, "is_active": true }

List Active Plans

GET /api/plans

Returns an array of all active subscription plans configured under the authenticated merchant project.

Response JSON (200 OK)
[ { "id": "e9c0b2cd-9e81-4d32-bc51-0a12e34f5678", "name": "Standard Monthly", "amount": 2000.00, "currency": "NGN", "interval_days": 30, "trial_days": 0, "is_active": true } ]

Retrieve a Plan

GET /api/plans/{plan_id}

Lookup details of a single plan by its unique ID.

Response JSON (200 OK)
{ "id": "e9c0b2cd-9e81-4d32-bc51-0a12e34f5678", "name": "Standard Monthly", "amount": 2000.00, "currency": "NGN", "interval_days": 30, "trial_days": 0, "is_active": true }

Delete a Plan

DELETE /api/plans/{plan_id}

Soft-deletes (deactivates) a billing plan. Existing active subscriptions on this plan will continue renewal cycles normally, but no new subscriptions can be created on this plan.

Response Code
204 No Content
