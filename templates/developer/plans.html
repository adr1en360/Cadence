API Reference

Plans API

Plans represent recurring billing schedules (such as monthly subscriptions, weekly services, or custom intervals) that you define and expose to your customers.

Create a Plan

POST /api/plans

Define a new billing package. The interval is defined explicitly in days (e.g., 30 for standard monthly).

Request Body (JSON)
{ "name": "Standard Monthly", "amount": 2000.00, "currency": "NGN", "interval_days": 30, "trial_days": 0 }
Response JSON (201 Created)
{ "id": "e9c0b2cd-9e81-4d32-bc51-0a12e34f5678", "name": "Standard Monthly", "amount": 2000.00, "currency": "NGN", "interval_days": 30, "trial_days": 0, "is_active": true }

List Active Plans

GET /api/plans

Returns an array of all active plans configured on your account.

Response JSON (200 OK)
[ { "id": "e9c0b2cd-9e81-4d32-bc51-0a12e34f5678", "name": "Standard Monthly", "amount": 2000.00, "currency": "NGN", "interval_days": 30, "trial_days": 0, "is_active": true } ]
